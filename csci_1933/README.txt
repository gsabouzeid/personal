These are projects from CSCI 1933 - Introduction to Algorithms and Data Structures
Took in the Fall of 2020 taught by Chris Dovolis

Projects include:
  1. A fractal drawer that can be made with 3 different shapes. Idea was to get familiar with OOP
  2. A battleship game that focuses on OOP and 2D arrays
  3. Comparing ArrayLists and LinkedLists and their time complexities
  4. A maze game that the computer solves using stacks and queues
  5. A program that creates a hash table with data distributed using an efficient hash function
